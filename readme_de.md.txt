# 🎾 racquelle — dein smarter Schlägerberater

**racquelle** ist ein minimalistisches, Bauhaus-inspiriertes Tennis-Schläger-Quiz,
das mithilfe eines kleinen Fragebogens deinen Spielstil erkennt
und dir eine Schlägerempfehlung gibt (z. B. von Tennis Warehouse Europe).

---

## 🧩 Projektstruktur

racquelle_demo/
│
├── index.html → Hauptseite
├── styles.css → Design (Bauhaus-Stil)
├── app.js → Quiz-Logik
├── questions.json → Fragen (DE/EN)
├── inventory.json → Schlägerdaten
└── README_DE.md → Diese Anleitung

---

## 🧠 So startest du lokal (offline)

1. Lade den Ordner **racquelle_demo** auf deinen Computer.  
2. Öffne die Datei **index.html** doppelklick — sie startet im Browser.  
3. Fertig 🎉  
   Du kannst das Quiz sofort spielen – es funktioniert auch **offline**.

---

## 🌍 Online stellen (gratis, kein Programmierwissen nötig)

### Variante 1 — GitHub + Vercel (empfohlen)

1. Gehe auf [https://github.com](https://github.com) → kostenlos registrieren.  
2. Klicke oben rechts auf **“New Repository”** → Name: `racquelle`.  
3. Lade alle Dateien (`index.html`, `app.js`, etc.) dort hoch.  
   - Du kannst jede Datei einfach per Drag & Drop ins Browserfenster ziehen.
4. Öffne [https://vercel.com](https://vercel.com) → ebenfalls kostenlos anmelden.  
5. Klicke **“Add New Project” → Import GitHub Repo** → wähle `racquelle`.  
6. Vercel baut deine Seite automatisch (kein Code nötig).
